## Core Principles

1. **Fully stateless** ? No runtime memory. State from EVENTS.log.jsonl
2. **Event sourcing** ? Append-only. sha256(idempotency)
3. **Config separated** ? All magic numbers in thresholds.yaml
4. **LLM isolated** ? Only for MCP parsing. Zero decision power
5. **Pure function core** ? regime.py, scoring.py have zero IO
6. **Orchestrator != Strategy** ? Zero business logic in runtime
7. **VETO hierarchy** ? Constitution > safety gate > hard veto > soft policy
